FunctionScript/FunctionScript on GitHub
Welcome to your very own FunctionScript Server example!

The current value of name is: name={%name}.
You can change the default behavior in ./functions/__main__.js

Play with the query parameter using this link:

./?name=Another%20Name.

You can see a simple addition API here:

./add/?a=1&b=2

You can see a simple SMS API here using Standard Library: utils/sms:

./sms/?tel=555-555-5555&body=Hello!
Try changing the phone number!
